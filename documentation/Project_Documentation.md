## Introduction

The E-Commerce Cart Logic project is a Node.js-based application designed to simulate the core operations of an online shopping cart system. The application demonstrates how products are managed, added to a shopping cart, validated against available stock, updated, and processed during checkout.

---

## Objectives

* Implement cart management functionality.
* Demonstrate business logic used in e-commerce systems.
* Apply stock validation techniques.
* Implement discount calculations.
* Create a modular Node.js application.

---

## Modules

### Product Module (products.js)

Stores product information including:

* Product ID
* Product Name
* Product Price
* Product Stock

### Cart Module (cart.js)

Responsible for:

* Adding products to cart
* Removing products from cart
* Updating quantities
* Viewing cart contents
* Calculating totals
* Applying discounts

### Checkout Module (checkout.js)

Handles:

* Order confirmation
* Payment summary
* Checkout completion

### Main Application Module (app.js)

Controls the overall flow of the application by coordinating all modules.

---

## Functionalities Implemented

### Add to Cart

Allows customers to add products to the cart.

### Remove from Cart

Allows customers to remove products from the cart.

### Update Quantity

Allows modification of product quantities within the cart.

### Stock Validation

Prevents customers from purchasing quantities beyond available stock.

### Discount System

Applies a 10% discount when the total purchase amount exceeds ₹50,000.

### Checkout

Displays final order information and confirms successful purchase.
